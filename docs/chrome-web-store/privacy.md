# Privacy Policy

LLM Output Guard is a local-only browser extension for detecting broken Markdown code fences in supported LLM web responses.

## Data Collection

LLM Output Guard does not collect, sell, transmit, or store user data.

## Local Processing

The extension reads visible assistant response text on supported LLM websites only so it can validate Markdown code fence structure. Validation and fixed-preview generation run locally in the browser.

## Network Requests

The extension does not make network requests and does not call AI APIs.

## Supported Sites

The extension runs only on:

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`
- `https://claude.ai/*`
- `https://gemini.google.com/*`

## Original Content

The extension does not replace or rewrite original assistant responses. It adds a local warning badge and a fixed-preview panel when Markdown fence issues are detected.
